# Calc
CLI Calculator

## Usage
calc <expression>

## Support
### Constants
- `pi` - 3.141592653589793
- `e` - 2.718281828459045

### Unary operators
- `-` - negation
- `abs` - absolute value
- `sin` - sinus
- `cos` - cosinus
- `acos` - inverse cosinus
- `asin` - inverse sinus
- `sqrt` - square root

### Binary operators
- `+` - addition
- `-` - subtraction
- `*` - multiplication
- `*-` - multiplication by negation
- `/` - division
- `rt` - root (2 rt a=sqrt(a))
- `^` - power
